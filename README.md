# 내 사이트 첫걸음

GitHub Pages 무료 주소부터 개인 도메인, DNS, HTTPS까지 안내하는 완전 초보자용 원스톱 웹 가이드입니다.

## 실행

`index.html`을 더블클릭하면 바로 사용할 수 있습니다.

## 자동 점검

PowerShell에서 `check-site.ps1`을 실행하면 파일명, CSS·JavaScript 연결,
화면 요소, 이미지 경로, 괄호, 주요 기능을 자동으로 확인합니다.

## 주요 기능

- GitHub 아이디와 도메인을 입력하면 주소·저장소·DNS 값 자동 완성
- 단계별 체크와 진행률 저장
- 첫 홈페이지 코드와 DNS 표 원클릭 복사
- 가입부터 HTTPS까지 8단계 상세 로드맵
- 404, www, 루트 도메인, HTTPS 문제 해결
- 모바일 화면 최적화 및 인쇄용 PDF
- 사용자가 제공한 4장 요약 인포그래픽

## 파일

- `index.html`, `styles.css`, `app.js`: 완성형 웹 가이드
- `GitHub-Pages-도메인-디테일-가이드북.pdf`: 공유·인쇄용 PDF
- `starter-site/`: 복사해서 쓰는 첫 홈페이지
- `assets/`: 단계별 요약 이미지
